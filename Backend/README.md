# VitalApp — Backend (MVP)

API REST mínima para VitalApp.

## Requisitos
- Java 17 + Maven 3.8+
- Backend en `http://localhost:8080`
- (Opcional) MySQL en Docker ya configurado, pero **no es requerido** para el MVP dummy.

## Correr en local
```bash
mvn clean package -DskipTests
mvn spring-boot:run
